# Minha Primeira Página Web

Este é um projeto simples criado para praticar os conceitos básicos de:

* HTML
* CSS
* JavaScript

## Funcionalidades

* Exibe uma mensagem na tela.
* Possui um botão interativo.
* Ao clicar no botão, o texto da página é alterado usando JavaScript.
